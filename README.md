# 🐭 Copa do Mundo dos Ratos

A Copa do Mundo dos Ratos é uma simulação em Python que reúne alguns dos ratos mais famosos da ficção em um torneio para descobrir quem seria o campeão em confrontos diretos.

Cada competidor possui atributos próprios, como vida, força, velocidade, resistência, precisão e inteligência, inspirados em suas características nas obras originais. Durante o torneio, esses atributos são utilizados para simular batalhas de forma automática.

Na fase de grupos, todos os confrontos são simulados. No entanto, no mata-mata, a simulação vale apenas 50% da nota. Os outros 50% estão em função de votação do público - fase ainda em desenvolvimento.

Este repositório contém os dados dos competidores, a lógica das batalhas e a execução completa do torneio, permitindo reproduzir os resultados ou criar novas edições da competição.

Os modelos matemáticos, lógicas e sistemas foram idealizados originalmente pelo dono do repositório. Contudo, os códigos foram gerandos com auxílio de IA, a fim de agilizar o desenvolvimento.
